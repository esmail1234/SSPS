# Statistical Analysis Platform

تطبيق ويب تفاعلي مبني بـ **Streamlit** لتحليل البيانات وعرضها بصريًا وإحصائيًا في مكان واحد.

الملف الأساسي في المشروع:
- [`spss.py`](/d:/SSPS/spss.py)

التطبيق الحالي مصمم بثيم فاتح مستوحى من لوحة التحكم في الصورة التي أرسلتها:
- خلفية بيضاء نظيفة
- أزرق رئيسي واضح
- لمسات mint/teal خفيفة
- بطاقات بيانات ناعمة وسهلة القراءة

## شرح عام للملف `spss.py`

### 1) الاستيراد

في بداية الملف يتم استيراد المكتبات الأساسية:

```python
import streamlit as st
import pandas as pd
import numpy as np
import plotly.express as px
from scipy import stats
```

### 2) الألوان والثيم

يحتوي الملف على palette ثابتة تمثل هوية التصميم:

- أزرق رئيسي للـ header والأزرار
- أزرق أغمق للتدرجات
- Mint للألوان المساعدة
- درجات فاتحة للكروت والخلفيات

### 3) إعداد الصفحة

```python
st.set_page_config(...)
```

هذا الجزء يحدد:
- عنوان الصفحة
- الأيقونة
- شكل التخطيط
- فتح الـ sidebar افتراضيًا

### 4) CSS مخصص

يتم حقن CSS داخل Streamlit لتعديل:
- الخلفية
- ألوان النص
- شكل الـ sidebar
- شكل البطاقات
- شكل الأزرار
- شكل صندوق رفع الملف

### 5) Hero Section

يوجد عنوان علوي كبير بعنوان:
- `Analysis Dashboard`

وهو يمثل واجهة البداية للتطبيق.

### 6) إدارة `session_state`

يتم حفظ:
- `df`
- `filename`

حتى لا تضيع البيانات عند أي تفاعل مع الواجهة.

### 7) رفع الملف من الـ sidebar

المستخدم يختار:
- مجال علمي
- ثم يرفع ملف `CSV` أو `Excel`

بعد الرفع يتم:
- قراءة الملف
- تخزينه في `session_state`
- إظهار رسالة نجاح أو خطأ

> ملاحظة: تم تعديل جزء رفع الملف ليكون واضحًا ومقروءًا داخل الـ sidebar.
